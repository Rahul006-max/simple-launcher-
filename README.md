# Minimal Text Launcher

A minimal, text-based Android launcher app focusing on typography, monochrome aesthetics, and zero visual clutter.

## Features

- **Typographic Clock**: Massive HH:mm display with date.
- **Battery Status**: Minimalist percentage display.
- **Favorites**: Quick access to your top 5 apps.
- **Minimal To-Do**: Integrated list to keep track of tasks without leaving the home screen.
- **App Drawer**: Clean, text-only app list accessible via swipe-up.
- **Monochrome Design**: Strictly black and white for maximum focus.

## Customization (Coming Soon)

- Change favorite apps.
- Customize text color and fonts.
- Set custom backgrounds/wallpapers.

## Installation

1. Clone the repository.
2. Open in Android Studio.
3. Build and deploy to your Android device.
4. Set as the default Home app in Android Settings.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: Android Native
